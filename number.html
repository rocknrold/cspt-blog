Binary

Application of binary digit in computing:

The binary is the most important number system in a computing machine because it is the only numbers that a computer understand as it is the lowest/smallest unit of data for the computer.

Binary digits – digits 0 and 1 – are used by computers to store data. A 0 or a 1. is represented by it. Binary numbers, such as the binary number 1001, are made up of binary digits (bits).

Computer programs are instruction sets. Simple binary codes which activate the CPU are translated into machine code for each instruction. Programmers write computer code and this is converted into binary instructions by a compiler that can be executed by the computer.
Binary storage is also used to store all software, music, documents, and any other information that a computer processes.

Octal

A fancy way to use the Octal number system on the computer is by this commonly used linux command which is the managing of file permissions.

Three commands control the permissions associated with a file:

  • chown user file changes the owner of the file;
  • chgrp group file alters the owner group;
  • chmod rights file changes the permissions for the file.

The (octal) numeric representation associates each right with a value: 4 for read, 2 for write, and 1 for execute. We associate each combination of rights with the sum of the figures. Each value is then assigned to different categories of users by putting them end to end in the usual order (owner, group, others).
