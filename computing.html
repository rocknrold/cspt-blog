What is the role of computing platforms?

Computing platforms comes in a various types and its usage in the industry and in the modern world were technology is in demand. In Information technology, computing platforms are used and has been a great help for almost all business and peoples lives such as developers.

Computing

Computing is the process of using computer technology to complete a given task and perform various operations such as structuring, managing and processing any kind of information.

This does not only refer to the using of computers but rather a refers to any kind of devices or machines that can perform operations on information and is running on top of platforms.

Platforms

A platform is a collection of tools for creating, modifying, maintaining, and monitoring services and goods in the digital world. Cloud infrastructures, APIs, user interfaces, and everything else that can be used for configuring, designing, creating, and managing products are examples of those platforms.

Combining two concept, there we have Computing Platforms

Computing Platforms

 A computing platform allows software to run on the computer and requires a processor, operating system, and programming environment with related runtime libraries and user agents.

Computing platforms are usually used by IT specialists, development agencies, developer groups or single developers. Developers who use such platforms include software developers, game developers, web developers, Software as a Service developer, Platform as Service developers and others.

Different kinds of computing platforms

  • Hardware Platforms
  • Operating Systems
  • Client/Server Platforms
  • Mobile Platforms
  • Cloud Platforms

On this topic we’re focusing on the Cloud computing platforms as it is also a popular type of computing platform. And a demand of jobs for cloud engineers are rising as it is also one of the path you can take after pursuing Information technology degree.

Cloud computing platforms is used for development, deployment and managing cloud infrastructure technologies such as software and services over the internet.

It offers wide variety of usage and purposes depending on the need of an individual using or in a business transitioning to use these kind of computing platforms. This cloud computing has been widely adapted in the business for the last decade up until now it is greatly improved. As it is a cost effective way to handle business need such as resources, and offers easy access and flexibility mainly for scalability, availability, maintainability and security for some types of cloud platforms.

Most common cloud computing platforms

Amazon Web Services : EC2, S3 BUCKETS

Google Cloud Compute Engines : Google Cloud Services

Microsoft Azure

The above mentioned are the most common and popular cloud computing platforms today. They offer large and multiple services that is suitable for the business needs.

Here is the service types/models cloud computing offers

Infrastructure as a service (IaaS)

The most basic category of cloud computing services. With IaaS, you rent IT infrastructure‚Äîservers and virtual machines (VMs), storage, networks, operating systems‚Äîfrom a cloud provider on a pay-as-you-go basis.

Platform as a service (PaaS)

Platform as a service refers to cloud computing services that supply an on-demand environment for developing, testing, delivering, and managing software applications. PaaS is designed to make it easier for developers to quickly create web or mobile apps, without worrying about setting up or managing the underlying infrastructure of servers, storage, network, and databases needed for development.

Software as a service (SaaS)

Software as a service is a method for delivering software applications over the Internet, on demand and typically on a subscription basis. With SaaS, cloud providers host and manage the software application and underlying infrastructure, and handle any maintenance, like software upgrades and security patching. Users connect to the application over the Internet, usually with a web browser on their phone, tablet, or PC.

Technologies that uses cloud computing

Docker

Docker is a software platform that simplifies the process of building, running, managing and distributing applications. It does this by virtualizing the operating system of the computer on which it is installed and running.

Kubernetes

Kubernetes (also known as k8s or “kube”) is an open source container orchestration platform that automates many of the manual processes involved in deploying, managing, and scaling containerized applications.

In other words, you can cluster together groups of hosts running Linux¬Æ containers, and Kubernetes helps you easily and efficiently manage those clusters.

The above technologies are all powered by cloud computing platforms. As seen above docker lets you manage your codes or other set of technologies into a container then this containers are then used by kubernetes as it is responsible for orchestrating and managing for application deployments.
