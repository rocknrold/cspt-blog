What is a computer hardware system?

When pertaining to a computer there are layers we should know and these include the computer hardware system which pertains to the most lower layer of the system which is responsible also for us to run our software and commands.

A computer hardware pertains also to the collections of components or physical parts of a computer including processor, ram, hard drives, solid state drive, power supply and etc., that is when combined can be called a computer hardware system.

Above is the example of computer architectural layer and shows how hardware becomes the base of a computer.

Choosing the right hardware for IT needs…

A good computer build is always based on the needs. Typically builds are done through requirement specification from a user or organization just like how software’s are built through SRS (System Requirements Specification).

As an example let’s say a user wanted to have his/her computer to be:

  • Needs to handle lots of multitasking and overloading
  • Has a decent GPU for faster rendering
  • Reasonable amount of ram that could handle multiple open task
  • Has an ssd, and plenty of hdd capacity for storage
  • Typically used for software developments and IT operations.
  • Lots of power for day to day operation!

The above list is for Philippine prices and these were based on the real computer store.

The list above is the components needed to build the wanted computer of our example user. In here the build is a little pricey because it will be used on a day to day.

The amount of computing resources using our custom build is enough to go multitasking and run applications used by the user with pretty high system requirements and some of the applications that can run on this build is:

Docker

Android Studio

Visual Studio Community or Equivalent IDE

Git and Github

Terminal

And the RAM eater

GOOGLE CHROME

It is a total package running these software’s on our custom build pc.
